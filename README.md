# Dataset Understanding and Basic Analytics

## Task Objective
To understand the structure of a dataset, explore data types, identify missing values,
and generate basic statistical insights using Python and spreadsheet tools.

## Tools Used
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Jupyter Notebook
- Excel / Google Sheets (comparison)

## Task Steps
1. Import libraries
2. Load dataset
3. View first records
4. Dataset structure
5. Statistical summary
6. Missing values check
7. Unique values check

## Outcome
- Dataset structure understood
- Data quality issues identified
- Summary statistics generated
